# langchain-mistral-ollama-chainlit
Simple Chat UI using Mistral model via Ollama, LangChain and Chainlit

### Open Source in Action 
- [Mistral](https://huggingface.co/mistralai/Mistral-7B-v0.1?library=true) as Large Language model via [Ollama](https://ollama.com/)
- [LangChain](https://www.langchain.com/) as a Framework for LLM
- [Chainlit](https://docs.chainlit.io/langchain) for deploying.
